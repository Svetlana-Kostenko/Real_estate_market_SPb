# Исследование объявлений о продаже квартир

Проект посвящён анализу данных «Яндекс Недвижимость» по продажам квартир в Санкт‑Петербурге и окрестностях для выявления ключевых факторов, влияющих на рыночную стоимость объектов. На основе пользовательских характеристик (площадь, количество комнат и др.) и геопространственных параметров (расстояние до центра, число парков и водоёмов поблизости) будет проведён исследовательский анализ, позволяющий определить закономерности ценообразования на рынке недвижимости.
